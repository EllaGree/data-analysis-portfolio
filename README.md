# Data Analysis Portfolio

A collection of data analysis projects built during an intensive 2-week learning sprint.

## Projects

### Python Analysis
- **Day 1** — Pandas Basics
- **Day 2** — Data Cleaning
- **Day 3** — Aggregations & Joins
- **Day 4** — Data Visualization
- **Day 5** — Full EDA Project (Restaurant Tips Analysis)
- **Day 6** — SQL + Python Integration
- **Day 7** — Visualization Review & Strengthening

### Portfolio Project
- **Day 10** — Sales Analysis Report (2,823 records across 25 columns)
  - Revenue trend analysis (2003–2005)
  - Product line performance breakdown
  - Geographic revenue distribution
  - Business recommendations

### Business Intelligence
- **Day 8-9** — Interactive Power BI Dashboard
  - Revenue by Month (Bar chart)
  - Complaints Trend (Line chart)
  - Revenue by Department (Pie chart)
  - Interactive Month Slicer

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook, Power BI, SQLite
